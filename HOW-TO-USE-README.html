How to Use Your Interactive Handbook

Getting Started

1 Opening the Handbook: Locate the file named "simple-handbook.html" in the "src" folder and double-click to open it in your web browser.
2 Navigation: Use the colored buttons at the top of the page to navigate between different sections of the handbook.
3 Interactive Elements: Throughout the handbook, you'll find clickable elements, drop-down menus, and interactive tools to enhance your experience.
Tip: For the best experience, use modern browsers like Chrome, Firefox, or Edge. Some features may not work properly in older browsers.

Updating Your Handbook

1 Receiving Updates: When you receive new content files, they will typically come as HTML files or folders with assets.
2 Adding New Files: To add new files:
  • Create a backup of your current handbook folder (just in case)
  • Drag and drop the new files into the appropriate locations:
    • HTML files usually go into the main folder or the "src" folder
    • Images go into "assets/images/" and its subfolders
    • Other resources (like scripts) typically go into designated folders
3 Replacing Files: When replacing files, your browser might show the old version due to caching. Press Ctrl+F5 (or Cmd+Shift+R on Mac) to perform a hard refresh.
Important: Always make sure file names match exactly what's expected. The handbook looks for specific file names in specific locations.

Special Features

Interactive Tools

The handbook includes several special interactive tools:

  • Legacy Weapons Creator: Design and customize legendary equipment
  • Regalia Maker: Create special ceremonial items
  • Fishing Minigame: A fun interactive fishing simulation

Filters and Search

Many sections include filter controls to help you find specific content quickly:

  • Use dropdown menus to filter by category
  • Click on tags to filter by specific attributes
  • Use the search box when available to find content by keyword
Tip: Click the "Reset" button to clear all filters and return to the full view of content.

Troubleshooting

1 Content Not Displaying: If parts of the handbook aren't showing:
  • Check that all files are in their correct folders
  • Try a hard refresh (Ctrl+F5)
  • Ensure your browser is up to date
2 Interactive Features Not Working: If buttons or interactive elements don't respond:
  • Make sure JavaScript is enabled in your browser
  • Check if any script blocking extensions might be interfering
  • Try opening the handbook in another browser
3 Images Not Loading: If images aren't appearing:
  • Verify that the image files exist in the assets/images directory
  • Check that the file names match exactly (including capitalization)
Need Help? If you continue experiencing issues, check the readme.txt file for contact information or additional troubleshooting steps.

Creating a New Content Section

If you want to add your own custom content section to the handbook:

  1. Create the HTML section: Add a new <div> with a unique ID to the handbook, following the same pattern as other sections:
    2. <div id="your-new-feature" class="content-section"></div>
  2. Add a navigation button: Include a new button in the navigation menu:
    3. <button id="btn-your-feature">Your Feature</button>
  3. Add JavaScript event handler: Create the event listener for your button:
    4. document.getElementById("btn-your-feature").addEventListener("click", function() {
  // Hide all content sections
  const contentSections = document.querySelectorAll(".content-section");
  contentSections.forEach(section => section.classList.remove("active"));
  
  // Deactivate all buttons
  const navButtons = document.querySelectorAll("nav button");
  navButtons.forEach(button => button.classList.remove("active"));
  
  // Activate your section
  document.getElementById("your-new-feature").classList.add("active");
  this.classList.add("active");
});
  4. Update the hideAllSections function: Add your section to the function:
    5. document.getElementById('your-new-feature').classList.remove('active');
Advanced: You can either add content directly or load it via an iframe from a separate HTML file, just like the Legacy Weapons and Regalia creators do.
